# e621.net Crawler

```
usage: e621Crawler [-h|--help] [-s|--start <integer>] [-c|--concurrency
                   <integer>] [-o|--output "<value>"] [-v|--verbose]

                   

Arguments:

  -h  --help         Print help information
  -s  --start        ID from where to start crawling. Default: 0
  -c  --concurrency  Concurrency. Default: 4
  -o  --output       Output folder. Default: Download
  -v  --verbose  
```
